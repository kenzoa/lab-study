### 十进制转二进制
for example：25
- 方案一
```math
25=2^4 + x^3 + x^0
```

所以25=11001B（B为二进制数的标志）</BR>
- 方案二</br>
25/2每次取余数，然后将其余数倒着写。
### 二进制转十进制
for example：11001B

```math
1*2^4 + 1*2^3+1*2^0=25
```
### 二进制转十六进制
for example：11001B</br>
拆分位八位：
```math
0001=2^0=1

1001=2^3+2^0=9
```
11001B=19H(H为十六进制标志)
### 十六进制转二进制
for example：19(H)</br>
拆分-原理同上
### 十进制转十六进制
for example：25
25/16每次取余数，然后将其余数倒着写。25=19(H)
